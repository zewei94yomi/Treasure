# 惊魂寻宝鸭 · 搜打撤 🦆

网页版本地双人寻宝游戏：俯视 2D + 视野/阴影系统，美术风格致敬鹅鸭杀，玩法参考搜打撤/撤离模式（蛋仔派对「惊魂寻宝队」、逃离鸭可夫）。

纯 HTML5 Canvas + 原生 JavaScript，**零依赖、零构建**。

## 🎮 在线试玩

**➡️ https://zewei94yomi.github.io/Treasure/**

电脑浏览器直接打开即可（键盘操作；同屏双人需两人共用一套键盘）。进度存在你自己浏览器的 localStorage 里，刷新/关闭都不会丢。
> 📱 目前是键盘操作，手机可以打开但暂不支持触屏操控。
> ⌨️ **请在英文输入法状态下游玩**——中文输入法会接管 `/` `。` `，` 等标点键位（游戏已内置全角字符回退映射与输入法检测提示，但英文态最稳）。练习场左下角会显示"最近按键"，按了没反应时可据此自查。

## 怎么玩（本地）

**推荐**：起个本地服务器（存档最稳）：
```bash
python3 -m http.server 8787
# 浏览器打开 http://localhost:8787
```
也可以直接双击 `index.html`（若浏览器阻止存档写入，主菜单会出现红色警告条）。

## 核心规则

- **撤离才算数**：开宝箱拿宝物，走到撤离点站 2 秒才能带出。带出的宝物换金币 + 收录进图鉴。
- **死亡惩罚**：阵亡时宝物散落原地（图鉴记为"目击"），武器、护甲、腰包**永久丢失**。
- **武器 ×16**：可带主副两把（R/L 切换）。枪有弹药/耐久/**有效射程**，命中**击退**；辅助瞄准 + 轻微弹道追踪。特色武器：影袭匕首（背刺 3 倍）、静音鹅弩（不引怪）、轰天雷（爆炸）、寒霜喷罐（减速）、**火舌喷灯（灼烧 DoT）**、**急冻线圈枪（冻结 1.3s）**、**鹅王·穿云（1400 射程超狙）**、矿脉鹤嘴镐（集齐「地心结晶」解锁图纸）。
- **怪物 ×10**（模型全面精细化：渐变材质/高光/动画细节）：幽影 / 疾爪蝠 / 石巨魁（免击退+熔核裂纹）/ 潜伏者 / 幽火（远程）/ 裂形泥怪（分裂）/ 尖啸者（召集）/ **骨戟卫兵（正面持盾格挡 65% 远程伤害，绕背打）** / **咒眼（凝视 1.8 秒即向全场揭示你的位置）** / 宝箱怪（35% 概率）。**攻击有前摇硬直**。
- **雇佣兵 ×3**：铁嘴保镖(600)/独眼老兵(1500)/佣兵王·灰羽(3200)，单局雇佣、跟随作战、替你挡刀；**每位各 2 次免费试玩**。
- **奖杯 ×12**：初出茅庐、影子行者（困难+全程未被发现）、无声杀戮（全背刺击杀）、地狱归来、冰湖舞者……主菜单「奖杯陈列室」查看。
- **系列集齐实物奖励**：每个宝物系列集齐除称号外还解锁**皮肤**（拾荒鸭/幽影/泽蛙蓑衣鸭/午夜神鸭）、**装饰**（海盗帽/船长帽/法老金辉/宝石虹光等 7 款光环帽子）或**武器图纸**（鹤嘴镐）。
- **难度 ×4**：简单 / 普通 / 困难 / **地狱**（25+ 只怪全种类，宝箱最豪华，独占宝物）。
- **地图 ×6**：遗忘庄园、幽暗矿洞、沉船湾、血月教堂（怪物视野+10%）、迷雾沼泽（玩家视野-18%）、冰湖洞窟（**冰面打滑**）。每张图 6 件**专属宝物**。
- **潜行**：按潜行键切换开/关（1P/2P 均为切换式）——无脚步声、被发现距离 ×0.30（**静止 ×0.18 近乎隐形**）、怪物背后盲区更大（×0.22）。**察觉蓄力**：被怪物注视需持续 0.7 秒才会暴露（头顶出现 👁 警告，有时间撤离）；困难/地狱的主动搜捕对潜行者只知道大概方位（±220px）。**翻滚、非静音枪开火、受击自动破隐**（静音鹅弩/近战不破隐）。隐身药剂 10 秒。
- **负重**：轻装 +8% 移速且安静；超载 -25% 移速、脚步震天响、更容易被看见。
- **护甲 ×3 + 腰包**：护甲按比例吸收伤害（可维修）；腰包 +3 背包格。
- **神秘商人**：困难/地狱出没，卖稀罕货，按 55% 价值收购宝物（不计图鉴）。
- **图鉴**：72 件宝物 × 11 系列；三态收集；未收录掉率 ×1.5；20 箱保底史诗。
- **双人分屏** + 共享小地图 + **底部背包状态栏**；怪物掉碎金（难度越高越多）、神话警报、BGM 心跳加速。
