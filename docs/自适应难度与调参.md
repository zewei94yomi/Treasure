# 自适应难度（DDA）与策划调参面板

## 自适应难度系统（无双割草）

**目标**：让强玩家更快遇到压力、弱玩家有喘息，全程贴着"心流"边缘。

**实现**（js/game.js `hordeUpdate`）：
- 每 4 秒采样一次存活玩家的**平均血量百分比**作为压力信号
- 目标强度 = `1 + (血量% − 55%) × 2 × 强度系数`，钳制在 `[1−系数, 1+系数]`（默认系数 0.15 → 0.85~1.15）
- 实际强度向目标做 50% 平滑插值，避免突变
- **作用点**：刷怪间隔 ÷ 强度（血多→刷更快）；狂暴出生概率 × 强度

满血速刷的玩家会感到怪潮明显变密；被打到 35% 血以下时系统悄悄松手。可在调参面板关闭（`自适应难度=0`）或调整强度（`自适应强度`）。

**设计权衡**：只用血量作信号（不用击杀速度）——血量直接反映"玩家当下的危险感"，且不会惩罚故意风筝不杀的玩法。

## 策划调参面板

主菜单 → 🎛️ 策划调参面板。20 个滑杆实时生效、持久保存（`SAVE.tuning`），恢复默认一键清空。

| 参数 | 默认 | 作用点 |
|---|---|---|
| 玩家移速/伤害/生命 | 1 / 1 / 1 | updatePlayer / tryAttack / 开局 |
| 怪物血量/伤害/移速 | 1 / **1.05** / **1.03** | 出生时 / damagePlayer / chase 速度 |
| 割草刷怪速率 | **1.05** | 刷怪间隔 |
| 经验/金币/磁吸/体力恢复 | 1 | 对应系统 |
| 宝箱怪概率 | 0.35 | 神秘箱开启 |
| 商人出现倍率 | **1.2** | 经典概率 & 割草出摊时机 |
| Boss 血量 | 1 | 割草 Boss |
| 翻滚冷却倍率 | 1（基准已降至 0.35s） | tryRoll |
| 割草时长(分钟) | 15 | 胜利判定与倒计时 |
| 荆棘强度 | 1 | 反伤 & 每级 1.5 减伤 |
| 自适应难度/强度 | 开 / 0.15 | DDA |
| 打击特效 | 开 | 伤害数字 + 打击停顿 |

加粗为本轮"稍稍调高一丢丢难度"的新基准。
